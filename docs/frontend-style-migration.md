# 前端高保真样式迁移记录

日期：2026-05-29

## 比对结论

### 原型新增能力

- 视觉系统：科技蓝、清透青、活力橙、柔和玫瑰色组成的年轻化高校服务配色。
- 布局风格：大圆角卡片、柔和弥散阴影、无边框页面区块、毛玻璃导航。
- 动效体验：页面 fade-up 入场、按钮按压回弹、卡片 hover 上浮、移动端 Tab 弹性激活。
- 展示能力：`web/prototype/index.html` 使用 iframe 展示四个独立手机原型页面，适合课堂演示。

### 生产前端保留逻辑

- 保留 `web/src/App.vue` 的哈希导航、角色权限过滤、远程 API 初始化、登录 / 退出、会话过期处理。
- 保留各业务模块的 Vue 组件、表单提交、文件上传下载、审批流、党团流程、通知统计和管理工作台逻辑。
- 不迁入原型中的静态模拟数据、iframe 展示板、独立 HTML 页面路由和 CDN Vue/Tailwind 运行方式。

## 本次迁移内容

- 将原型设计系统迁移到生产全局样式 `web/styles.css`。
- 为生产应用加入现代化浅色 / 深色模式变量、玻璃侧栏、玻璃顶栏、卡片层次、表格与表单统一风格。
- 在 `web/src/App.vue` 为当前页面组件增加 `view-stage` 包裹层，用于页面切换入场动画，不改变组件业务逻辑。
- 在 `web/src/main.js` 增加全局按压水波纹反馈，仅作用于按钮、导航和可点击卡片，不改变业务事件。
- 优化首页信息架构展示：状态卡使用 `metric-card`，核心功能入口使用 `feature-card` 与编号标识，提升可扫读性。
- 保留 `web/prototype/` 作为高保真展示与后续视觉参考。
